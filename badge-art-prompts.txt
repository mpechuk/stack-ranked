STACK RANKED — COMPLIANCE BADGE ART GENERATION PROMPTS
=======================================================
Generated from badges.json. One image-generation prompt per Compliance Badge —
i.e. one per Mandatory Training card (see the `trainings` array in cards.json).
Each Mandatory Training awards its badge when resolved; this file's art fills
the `image` field of the matching entry in badges.json and is dropped into the
center of each punch-out token by generate_badges.py (docs/Stack_Ranked_Badges.pdf,
6 tokens per badge type). Every prompt is preceded by a REFERENCE line
(badge name, source training, and grant count from badges.json) so you can match
generated art back to the right badge.

Unlike the wide banner art in card-art-prompts.txt, these are CENTERED CIRCULAR
EMBLEMS — think enamel lapel pin / embroidered corporate-compliance patch / wax
seal — because they sit inside a round token, not across the top of a card.

MASTER STYLE (baked into every prompt below, so each block is self-contained
and can be pasted into an image model on its own):
  Retro corporate-propaganda merit-badge emblem, matching the box cover
  illustration — a single centered circular seal (enamel pin / embroidered
  patch / wax seal), bold black ink outlines, halftone screentone shading, worn
  paper-grain texture, a subtle radiating sunburst inside the ring, one
  satirical heroic icon at the dead center, framed by a clean beaded/rope inner
  ring. Base palette of burnt orange, cream, and navy, lifted with the badge's
  accent color (see each block). No text, no lettering, no numerals, no logos,
  no watermarks — the badge's name and label are printed separately by the token
  layout; this is the icon-emblem only.

FORMAT: square 1:1 crop, the round emblem centered and filling ~80% of the
frame, on a plain flat background (no scene, no environment) so it drops cleanly
into a circular token. See IMAGE inner-disk placement in generate_badges.py.

Each entry below has a REFERENCE line (from badges.json) and a PROMPT you can
paste as-is into an image-generation model.
